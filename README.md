# ☁️ AWS Failover Routing Project

## 📌 Overview

This project demonstrates the implementation of a failover routing policy using AWS services to ensure high availability and reliability of applications.

## 🛠️ AWS Services Used

* Amazon Route 53
* Amazon EC2
* Health Checks

## ⚙️ Architecture

* Primary server (EC2 instance)
* Secondary backup server (EC2 instance)
* Route 53 failover routing
* Health check monitoring

## 🔄 How It Works

1. Users access the application via Route 53 DNS
2. Traffic is routed to the primary EC2 instance
3. Health checks monitor the primary server
4. If the primary fails, traffic is redirected to the secondary server

## 📊 Key Features

* High availability
* Automatic failover
* Improved system reliability
